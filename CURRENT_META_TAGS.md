2. Select "View Page Source"
3. Look for `<meta>` tags in the `<head>` section

### **Method 2: Browser Developer Tools**
1. Press F12 (or Cmd+Option+I on Mac)

3. Find `<head>` section
4. Expand to see all meta tags
